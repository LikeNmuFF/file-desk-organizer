What it does

Everything your file manager wishes it could do.

📂
01

Smart Scanning

Walks any folder tree instantly. Detects file types, sizes, and modification dates — even finds possible duplicates hiding in plain sight.

🏷️
02

Auto-Categorization

Files sort themselves into PDFs, Spreadsheets, Images, Code, and more. Smart tags pick up on finance, legal, HR, and report keywords automatically.

🤖
03

AI Companion

Ask questions about your files in plain English. Powered by Groq — blazing fast, runs through your API key, data stays on your machine.

⚡
04

One-Click Organize

Hit the button and every file moves into its category folder. Subfolders for tags like Finance or Legal. Done in seconds.

🔒
05

100% Local

Zero network calls for core features. No telemetry, no analytics, no cloud dependency. Your files never leave your computer.

📡
06

WiFi Sharing

Scan files from your phone or another device on the same network. Password-protected. Great for quick remote access without the cloud.

How it works

Three steps. That's it.

🗂️

Pick a folder

Click "Choose Folder" and point it at whatever mess needs sorting — Downloads, Desktop, that one folder you've been avoiding.

👀

Review the plan

See every file categorized and tagged. Search by name, extension, or date. Toggle category tabs. Spot duplicates highlighted in rust.

✅

Organize

Hit "Organize Files" and watch it work. Files move into clean category folders — PDFs/, Spreadsheets/Finance/, Images/ — instantly.

Your files. Your machine.

The Filing Desk is built with Tauri — a fraction of Electron's size, with a Rust backend that's fast and secure. Core features need zero internet. The AI companion uses your own API key. Nothing phones home.
