# deskmatee-beta v1.0.0

Your files, finally organized.

A desktop file organizer that scans, categorizes, and moves your files into place — locally, privately, on your machine. No uploads. No cloud. Just order.

## What's New

- **Smart folder scanning** — walks any folder and catalogs every file by type, size, and date
- **Auto-categorize** — sorts files into PDFs, Images, Documents, Audio, Video, Spreadsheets, Code, Archives, Fonts, and more
- **Smart tags** — detects Finance, Work, Personal, and Project files by content and keywords
- **Duplicate detection** — flags files with matching names and sizes
- **One-click organize** — moves files into a clean folder structure with a single click (or preview a dry run first)
- **AI Companion** — chat with an AI that knows your file library; ask it to find files, suggest organization, or answer questions (powered by Groq, bring your own API key)
- **WiFi Sharing** — share your organized folder with any device on the same network; password-protected, no internet required
- **~10 MB** — built with Tauri + Rust, a fraction of Electron's size

## System Requirements

- Windows 10 or later
- ~10 MB disk space
- Internet connection required only for AI companion (optional)

## Built With

- [Tauri v2](https://v2.tauri.app/) — Rust backend, web frontend
- [Groq](https://groq.com/) — fast AI inference (free tier available)
- Vanilla HTML/CSS/JS — no framework overhead
